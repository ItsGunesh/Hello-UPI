Hello-UPI is a vouce assisted payment system that uses voice commands as input to make a transaction after a small authentication.This is a quick and easy way to make payments without hassle of searching names and clicking multiple buttons to make a small transaction.   

Currently supporting 8 indian languages including English,Hindi,Marathi,Tamil,Telugu etc . The language can be activated using a simple greeting like hello in english in that particulat language.You can also use the already adopted QR method to make payments .  

The authentication includes pin and face recognition.  

Using advanced NLP methods any commands that resembles a transaction to a person is accepted in the Webapp.  

Tech Stack used React,Express,Node and FastAPI for website.The face recognition part is implemented using deepface library.The NLP part is implemented using re and spacy libraries.
